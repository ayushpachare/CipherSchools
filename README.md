# AI-Powered SQL Practice Platform.
An interactive SQL learning platform where users can practice writing SQL queries and get instant evaluation — enhanced with AI-generated hints using Google Gemini.

# Features
1 Practice SQL questions based on difficulty (Easy / Medium / Hard)
2 Real-time query evaluation
3 Actual vs User result comparison
4 AI-powered hints (Gemini API)
5 Dynamic question switching
6 Beginner-friendly learning flow

# Tech Stack

# Frontend
- React.js
- Tailwind CSS
- React Router

# Backend
- Node.js
- Express.js
- MongoDB

#AI Integration
- Google Gemini API

# How It Works
1. User selects difficulty level.
2. Writes SQL query based on the given table.
3. Platform shows:
   - Correct / Incorrect feedback
   - Expected result table
   - AI-generated hint (if needed)

# AI Hint System
Instead of revealing answers, Gemini:
Guides users  
Suggests clauses (WHERE / AND / OR)  
Helps improve logic  
This makes learning more interactive.
